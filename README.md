## 🛠 習得・活用テクニック

### 1. レイアウトとタイポグラフィの制御
- **Flexboxによる配置制御**: `align-items: center;` を用いた垂直方向の整列や、`column-reverse` によるスマホ表示時の要素順序の入れ替え（逆順レイアウト）を習得。
- **相対単位 `rem` の導入**: ルート（`html`）のフォントサイズを基準にサイズを指定し、レスポンシブ時の管理を効率化。
- **Google Fontsの活用**: `Crimson Text` を導入し、Webフォントを使用した洗練されたタイポグラフィを実装。

### 2. 適切な命名と要素選択（Markup & Naming）
- **詳細度（Specificity）の理解**: クラス名の付け方や要素の親子関係によるCSSの優先順位を意識し、意図したスタイルを確実に反映させる設計。
- **意味のある要素選択**: リンク（`mailto:`）の適切な記述方法など、役割に応じたマークアップを習得。

## 📝 自己評価・気づき
- **スピード**: 1時間という制限時間内で、HTML構造からスタイリングまで完結。模写は正確さとスピードの両立が重要であると再認識した。
- **コード品質**: 命名や要素の親子関係を整理したことで、見通しの良いコードになった。
- **今回の学び**: 
    - `column-reverse` を使うことで、HTMLの構造を維持したままデザインの要望に合わせて並び替えができる利便性を実感。
    - メディアクエリ内でのフォントサイズの上書き漏れなど、細部への注意力が精度に直結することを学んだ。
- **今後の課題**: 
    - 知識をさらに増やし、45分、30分と「迷わず手を動かす」レベルまで精度とスピードを高めていきたい。
    - 今回導入した `rem` の感覚をさらに磨き、より柔軟なレスポンシブ対応を目指す。
